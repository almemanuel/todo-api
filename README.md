# Todo API

This is a simple API project built with Node.js for study purposes. It demonstrates the basics of creating and managing a RESTful API.

## Features

- Create, read, update, and delete todo items
- Simple and clean codebase for learning

## Getting Started

1. **Clone the repository:**
    ```bash
    git clone https://github.com/almemanuel/todo-api.git
    cd todo-api
    ```

2. **Install dependencies:**
    ```bash
    npm install
    ```

3. **Start the server:**
    ```bash
    npm start
    ```

4. **API Endpoints:**
    - `GET /todos` - List all todos
    - `POST /todos` - Create a new todo
    - `PUT /todos/:id` - Update a todo
    - `DELETE /todos/:id` - Delete a todo

## Purpose

This project is for learning how APIs work with Node.js. Feel free to explore, modify, and experiment!
